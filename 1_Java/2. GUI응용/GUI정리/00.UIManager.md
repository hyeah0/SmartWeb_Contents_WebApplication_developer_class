# UIManager class

    - java.lang.Object
        - javax.swing.UIManager

- UIManager 는 현재 룩앤필, 사용 가능한 룩앤필 세트 , 룩앤필이 변경 될 때 알림을받는 PropertyChangeListeners , 룩앤필 기본값, 다양한 기본값을 얻기위한 편리한 메소드를 관리하는 클래스이다.

## - Mac gui button background color 지정하는 방법

- UIManager 클래스 setLookAndFeel 메서드 이용

- 형태
  - 메서드 안에 try catch 문
  - try 안에 적용하고 싶은 테마의 주소값 스트링 값으로 지정
  - UIManager.setLookAndFeel(지정한 주소값);

```
void gui_theme() {
		try {

			String lookAndFeel_nimbus = "javax.swing.plaf.nimbus.NimbusLookAndFeel";
			UIManager.setLookAndFeel(lookAndFeel_nimbus);

		}catch(Exception e) {
			e.printStackTrace();
		}
	}
```

- look and fell 목록 확인

```
	void gui_theme_check() {
		for(LookAndFeelInfo info : UIManager.getInstalledLookAndFeels()) {
			System.out.println(info.getClassName());
		}
	}
```

### - 주소값

- 디폴트 (운영체제 환경에 맞는 ui로 설정)

  - UIManager.setLookAndFeel(UIManager.getCrossPlatformLookAndFeelClassName());

- Metal \_ 버튼 색상 적용 가능

  - "javax.swing.plaf.metal.MetalLookAndFeel";

- Nimbus - 버튼 색상 적용 가능 (버튼 둥글하게\_borderline 재지정 필요)

  - "javax.swing.plaf.nimbus.NimbusLookAndFeel";

- Motif - 버튼 색상 적용 가능 (버튼 클릭시 빨강 테두리)

  - "com.sun.java.swing.plaf.motif.MotifLookAndFeel";

- Mac - 버튼 색상 적용 불가능

  - "com.apple.laf.AquaLookAndFeel";

- Windows (안됨)

  - "com.sun.java.swing.plaf.windows.WindowsLookAndFeel";

- Windows Classic (안됨)

  - "com.sun.java.swing.plaf.windows.WindowsClassicLookAndFeel";

- Mac (안됨\_상단 맥은 가능)

  - "com.sun.java.swing.plaf.mac.MacLookAndFeel";

- GTK (안됨)

  - "com.sun.java.swing.plaf.gtk.GTKLookAndFeel";
