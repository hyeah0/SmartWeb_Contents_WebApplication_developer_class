## 데이터베이스

- 데이터베이스는 실제 정보가 아니라, 아래와 같은 데이터 정보들을 여러개 가지고 있는 일종의 그릇

<table>
    <tr>
        <th>작성</th>
        <th>설명</th>
    </tr>
    <tr>
        <td>show databases;</td>
        <td>데이터베이스 목록 보기</td>
    </tr>
   <tr>
        <td>CREATE SCHEMA `데이터베이스명` DEFAULT CHARACTER SET 인코딩 DEFAULT COLLATE 데이터정렬방식;</td>
        <td>데이터베이스 생성</td>
    </tr>
    <tr>
        <td>drop database [데이터베이스명];</td>
        <td>데이터베이스 삭제</td>
    </tr>
    <tr>
        <td>use [데이터베이스명];</td>
        <td>데이터베이스 선택</td>
    </tr>
</table>
- 예약어는 소문자로 작성해도 되지만, 가급적 대문자 작성하는 것이 좋다.
- 사용자가 직접 만든 이름과 구분하기 위함

## - 데이터 베이스 생성 및 DB선택

<small>- [터미널 접속방법 참고](https://github.com/hyeah0/SmartWeb_Contents_WebApplication_developer_class/blob/main/0_MacSet/mysql/00_%ED%84%B0%EB%AF%B8%EB%84%90%EC%97%90%EC%84%9Cmysql%EC%A0%91%EC%86%8D%EB%B0%A9%EB%B2%95.md)</small>

<img src="https://github.com/hyeah0/SmartWeb_Contents_WebApplication_developer_class/blob/main/0_MacSet/mysql/image/terminal/createDatabase.png" width="100%">

```
CREATE SCHEMA `데이터베이스명` DEFAULT CHARACTER SET 인코딩 DEFAULT COLLATE 데이터정렬방식;

CREATE SCHEMA `nodejs` DEFAULT CHARACTER SET uft8mb4 DEFAULT COLLATE utf8mb4_general_ci;

use nodejs;

>>> [nodejs : 데이터베이스명]
```

- ⭐️ 인코딩 : utf8 >> 한글 / utf8mb4 >> 한글 + 이모티콘
- ⭐️ DB정렬방식 : utf8_general_ci; >> 한글 / utf8mb4_general_ci; >> 한글 + 이모티콘
